Hexo Blog 架設過程

在 Windows 下用 VirtualBox 架設一台 Ubuntu Server (ver.14.04)

然後再將 Hexo 裝在 Ubuntu Server 上，最後透過 Hexo 將 Blog push 上去

將之前專題用的的虛擬機技巧：轉 port
便可以在 Windows 下可以連到 Ubuntu Server
目的有：

  1. 在 Windows 下可以透過瀏覽器看到 Hexo server 開啟時的臨時網站的樣貌
  2. 建立 Windows 與 Ubuntu_Server 的溝通管道(手續很複雜…..根本自找麻煩)

Problems：

  1. 沒有 SSH key 無法成功 commit
  2. Ubuntu_Server 下的 ssh 公鑰無法被複製到 windows 下
  3. Git global 設定有問題導致 SSH key 配對錯誤，進而 commit 失敗
  4. CSS 遺失

Solutions：

1 ~ 3 其他文章有解法

CSS 遺失問題，由於是因為在 Blog 創建前，_config.yml 內的配置沒有設定好，
index 的 CSS 連結跑掉，導致在 github.io 上看到時，整個走樣，由於還有其他參數還沒改，
所以決定整個資料夾砍掉從新配置。

另外由於阿詠在配置時 .gitignore 是後來才創建，導致 _config 沒有被隱藏反而上傳了。
