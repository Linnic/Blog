Github SSH key for Ubuntu_Server on VirtualBox

一直沒搞懂 SSH key 要幹嘛，邊做邊發現有些問題，並回想之前在 Windows 上操作的情況，
然後才弄懂 Github 上 SSH keys 是在管理不同裝置的連線認證，新增的部分包含了’title’、’key’，
本以為 key 的部分只要貼中間亂碼的部分，後來才發現要完整的內容(包含其他參數)才行。

Problems：

  1. Ubuntu_Server 下，SSH key 無法直接被複製到 Windows 上
  2. PuTTY 無法直接抓檔(有其他解法不過沒用……)
  3. PSFTP 無法進入 ~/.ssh 抓 SSH key

Solutions：

在 Ubuntu_Server 上先將存有 SSH key 的檔案複製到其他資料夾，
在透過 SFTP 將檔案抓回本地端。
然後將得到的 key 放進 GitHub 上新增的 SSH key 內，就完成了。
