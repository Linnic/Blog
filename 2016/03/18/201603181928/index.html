Ubuntu Server 砍掉重練

由於 Hexo 改了一堆設定依舊一直無法嵌入 Disqus，加上一直重複安裝 nvm、npm、hexo、nodejs…..，
導致 Ubuntu Server & Hexo 呈現半癱的狀態，加上 tty 介面也用的膩了，所以決定重裝。

這次決定在 Ubuntu Server 上加入圖形化介面，下 sudo apt-get install ubuntu-desktop 來進行安裝桌面，
約 1.5G 等了滿久，再裝 gnome sudo apt-get install gnome-core，之後重新開機就進入到全彩的的世界了。

Problems:

  1. Hexo 的 NVM 安裝教學抓不到東西，所以失敗
  2. NVM 的 Script install 安裝成功，.nvm/.bash 無法寫入(權限問題)
  3. NVM 的 Script install 安裝成功，但因只有 tty，terminal 無法重啟
  4. Hexo 的 relative_link 設為 true 導致 Server 生成失敗
  5. Hexo 生成新的檔案，然後用將舊有的檔案用 SFTP 去取代新生成的檔案，導致文章產生錯誤，無法顯示網誌

Solutions:

  1. 改採官網的教學參考
  2. 用 chmod 777 ~/.bash or chmod a+rwx
  3. 加上 GUI 後，人生是彩色的~~
  4. 改為 false 即可
  5. 改採用複製內容的方式，再回傳
