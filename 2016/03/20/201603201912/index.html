python 編譯注意事項

python 的編譯器對縮排真的滿嚴格的~~
之前上課操作時沒發現到，沒處理到註解的部分 XD

註解的可分為單行註解：#，或多行註解：''' '''、""" """ 。

Problems：

  1. line 25 “”” 出現 IndentationError: unexpected indent
  2. from: can’t read /var/mail/mininet.link
    Syntax error: “(“ unexpected (expecting “then”)

Solutions：

  1. 一直沒弄懂為甚麼同樣是用 """ """ 來註解，第25行會錯，
    以為是 code 打錯，後來爬了一下文章，原來是 python 的嚴謹風格所致。

  2. 另外有關 docstring 位置的順序放置，在這篇文章中有詳細敘述。

    import 應該永遠放在檔案頂端，也就是在 module 的註解和 docstring 之後，而在 module 的全域變數和常數之前。
