Git Setting

終於只差最後一步，網站就能盛放上 Github。

Problem：

由於最後 commit 時，發生 global 下的 user.name、user.email 為預設，
與 SSH key 不匹配進而導致，commit 不成功，所以只要將這兩個參數設好，
就可成功將 project 上傳成功。

Solution：

參數設置：
$ git config --global user.name "Your name"
$ git config --global user.email "Your email"
